# A3idental React App

A modern, dynamic dentist website built with React + Vite. Blue/black theme, responsive layout, image slider, and multi-page navigation.

## Scripts
- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`

## Features
- React Router pages: Home, Services, About, Contact
- Responsive Navbar with mobile menu
- Attractive blue/black theme via CSS variables
- Hero, Services grid, Prosthetics image slider (autoplay), Gallery
- Contact form UI (ready to integrate)

## Deploy
- Netlify/Vercel: Framework = Vite/React. Build = `npm run build`. Output = `dist`.
- GitHub Pages: `npm run build` then publish `dist`.
